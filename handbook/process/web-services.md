# FreeCAD web services

This page lists the services controlled by the FPA, and who to contact in case of problems

### Self-hosted

| Service | Description                                                                                                                               | Address                   | Admins                                  |
|:------- |:----------------------------------------------------------------------------------------------------------------------------------------- |:------------------------- |:--------------------------------------- |
| Website | The main entry point and FreeCAD website. Code is at https://github.com/FreeCAD/FreeCAD-Homepage and pushed manually by one fo the admins | https://freecad.org       | kkremitzki, chennes, yorik              |
| Forum   | The FreeCAD forum, using phpBB software with a few extensions (translations, voting, mentions) and custom theme                           | https://forum.freecad.org | kkremitzki, yorik, chennes, Roy_043     |
| Wiki    | The documentation site of FreeCAD, using MediaWiki software with a series of extensions, and a git bridge                                 | https://wiki.freecad.org  | kkremitzki, chennes, yorik, reqrefusion |
